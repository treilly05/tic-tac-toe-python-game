# Python Tic Tac Toe Game

An interactive two-player Tic Tac Toe game built in Python using the Turtle graphics library.

---

## Project Overview

This project is a fully functional Tic Tac Toe game that allows two users to play against each other in a graphical interface created with Python Turtle. Players can enter their names, click on the board to place their moves, track scores across rounds, and replay the game after each match.

The project is split into two Python files:

- One file contains the core Tic Tac Toe game logic
- The second file handles the graphical user interface and player interaction

---

## Files Included

### `Final_Project_Game.py`

Main game execution file that:

- Creates the Turtle graphics window
- Draws the Tic Tac Toe board
- Handles mouse clicks
- Displays player turns
- Updates the scoreboard
- Detects rematches
- Draws the winning red line

### `Final_Project_Class_TicTacToe.py`

Contains the main Tic Tac Toe game logic including:

- Board creation
- Valid move checking
- Turn switching
- Winner detection
- Draw detection
- Board reset functionality
